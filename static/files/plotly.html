Plotly Visualization

Interactive data visualization

Usage:

This page automatically loads a Plotly JSON file. To use it:

  1. Save your R Plotly visualization as JSON (see R code examples below)
  2. Upload the JSON file to static/files/ directory
  3. Access via: plotly.html?json=your-figure.json
  4. Or rename your file to plotly-figure.json for default loading
